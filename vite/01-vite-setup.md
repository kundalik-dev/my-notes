# Introduction

Vite project setup

## Why vite

- Fast HMR

> Vite is not a framework.

## Setup

To set up project We use command `npm create vite@latest` or `pnpm create vite@latest`

```npm
 npm create vite@latest

// Framework -> React

// Variant -> JS, TS, JS + SWC, TS + SWC

// Create vite and run -> Yes
```
